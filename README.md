# fixable
Build fixable scale page on mobile platform
## Usage

默认设计稿750px,通过计算屏幕宽window.screen.width和设计稿的比例动态修改vieport的缩放达到页面的等比缩放，不需要rem，默认字号推荐设置24px以上。

#### Import

```html
<script src="/fixable.min.js"></script>
or
<meta name="viewport" view-width='640'>
<script src="/fixable.min.js"></script>
```
